제출파일 설명

- Images 폴더: 각종 이미지 파일들

1. Photos 폴더: 각 손글씨 이미지 인증사진 파일
2. Inference 폴더: 원본이미지와 숫자부분만 자른 이미지 파일(원본이미지 8장 + 자른이미지 8장 = 총 16장)
3. MNIST_test2 폴더: 모델 업로드한 이미지
4. MNIST_normalized 폴더: 모델 training 직전의 이미지 파일
5. sample_images: 처음에 몇번 시도해보았지만 실패한 이미지들

- 해방일지 관련 파일들

5. MNIST.ipynb : 원본 코드 파일
6. MNIST_project.pth: 모델 파일
7. Final Project.docx: 해방일지 문서 워드파일
8. Final Project.pdf: 해방일지 문서 pdf파일
9. references.docx: ChatGPT 사용 출처 정리 파일
